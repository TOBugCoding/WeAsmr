# WeAsmr-客户端ASMR播放器
# 一、简介
  WeAsmr收录ASMRMOON ASMRGAY等网站的音频资源，提供音声、视频的在线下载，支持播放列表的自定义，以极简的风格降低客户端操作难度，
  软件使用qt6.9.3,vlc4.0进行开发，支持WIndows10/11。
